#### 交付物一：`common.h`（全员依赖，最先完成）

所有结构体、宏常量、函数声明集中在此文件，其他人 `#include "common.h"` 即可使用。

**结构体定义**

```c
// 图书结构体
typedef struct {
    int bookId;
    char bookName[50];
    char author[30];
    char category[20];
    int stock;
    int borrowCount;
} Book;

// 图书顺序表
#define MAX_BOOK 200
typedef struct {
    Book data[MAX_BOOK];
    int length;
} BookList;

// 读者结构体
typedef struct {
    int readerId;
    char readerName[20];
    char gender[5];
    char phone[15];
    int role;   // 1=管理员 2=教师 3=学生
} Reader;

#define MAX_READER 200
typedef struct {
    Reader data[MAX_READER];
    int length;
} ReaderList;

// 借阅记录结构体
typedef struct {
    int borrowId;
    int bookId;
    int readerId;
    char borrowTime[15];   // 格式：YYYY-MM-DD
    char returnTime[15];   // 未归还填 "未归还"
    char deadline[15];     // 格式：YYYY-MM-DD
} Borrow;

#define MAX_BORROW 300
typedef struct {
    Borrow data[MAX_BORROW];
    int length;
} BorrowList;
```

**全部函数声明**（写在 common.h 底部，各模块实现）

#### 交付物二：`fileio.c / fileio.h`（全员依赖，与 common.h 同批交付）

| 函数 | 功能说明 |
|------|---------|
| `int LoadData(BookList *bL, ReaderList *rL, BorrowList *brL)` | 程序启动时从 book.txt / reader.txt / borrow.txt 读数据到内存；文件不存在则自动创建空文件；成功返回1 |
| `int SaveData(BookList *bL, ReaderList *rL, BorrowList *brL)` | 程序退出时将内存三张表覆盖写入对应 txt 文件；成功返回1 |

**文件格式约定（统一给全组）：**
- 每行一条记录，字段之间用英文逗号 `,` 分隔
- 日期统一格式 `YYYY-MM-DD`，未归还填字符串 `未归还`
- 读取时用 `sscanf` + `%[^,]` 分割字段，去除首尾空格
- book.txt 示例行：`1001,C语言程序设计,谭浩强,计算机,10,8`
- reader.txt 示例行：`2001,张老师,男,13800138000,2`
- borrow.txt 示例行：`3001,1002,2001,2026-06-01,未归还,2026-07-01`

#### 任务清单二：`borrow.c / borrow.h`（借阅/归还/查询/排行/提醒全部在此）

**借阅核心：**

| 函数 | 功能说明 |
|------|---------|
| `void AddBorrow(BorrowList *L, BookList *bL, Borrow br)` | 新增借阅记录；校验图书存在+库存>0；调用 `UpdateStock(-1)` 和 `UpdateBorrowCount()` |
| `int ReturnBook(BorrowList *L, BookList *bL, int borrowId, char *retTime)` | 归还图书；更新归还时间；调用 `UpdateStock(+1)`；返回1/0 |
| `void SearchBorrowByBook(BorrowList *L, int bookId)` | 按图书编号查该书所有借阅记录 |
| `void SearchBorrowByReader(BorrowList *L, int readerId)` | 按读者编号查个人借阅记录 |

**借阅统计排行：**

| 函数 | 功能说明 |
|------|---------|
| `void BorrowRank(BookList *bL)` | 将图书顺序表复制一份副本，对副本按 `borrowCount` 降序冒泡排序，格式化输出排行（不改原表顺序）|

冒泡排序逻辑：
```
外层循环 i = 0 到 length-2
    内层循环 j = 0 到 length-2-i
        若 data[j].borrowCount < data[j+1].borrowCount
            交换 data[j] 和 data[j+1]
```

**临期/超期提醒：**

| 函数 | 功能说明 |
|------|---------|
| `void CheckRemind(BorrowList *L)` | 遍历所有 returnTime=="未归还" 的记录；将 deadline 和当前日期各拆成年/月/日整数；剩余天数≤3输出临期，已超期输出超期 |

日期差计算方法：
- 获取今天：`#include <time.h>`，用 `time()` + `localtime()` 取年月日
- 解析字符串：`sscanf(deadline, "%d-%d-%d", &y, &m, &d)`
- 换算成总天数做差（year×365 + month×30 + day，精度满足本系统需求）

**子菜单入口：**

| 函数 | 功能说明 |
|------|---------|
| `void BorrowManage(BookList *bL, ReaderList *rL, BorrowList *brL, int role)` | 借阅业务总入口，根据 role 控制菜单项：管理员显示全部，教师/学生仅显示借阅/归还/个人记录/提醒 |
