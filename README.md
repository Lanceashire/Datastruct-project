# 高校图书管理系统

一个使用 C 语言和顺序表实现的控制台图书管理系统，用于数据结构课程综合实验。

系统支持图书、读者和借阅记录管理，并使用本地 TXT 文件保存数据。管理员拥有完整管理权限，教师和学生可查询图书、办理借还、查看个人记录及借阅提醒。

## 功能

- 图书信息新增、删除、修改、编号查询、书名模糊查询
- 读者信息新增、删除、修改和查询
- 图书借阅与归还，自动联动库存和累计借阅次数
- 按图书或读者查询借阅记录
- 按累计借阅次数生成图书排行
- 临期三天和超期借阅提醒
- 管理员、教师、学生三级菜单权限
- 图书、读者、借阅数据自动加载与保存
- TXT 文件按行查看、插入、删除、替换和关键字搜索

## 数据结构

系统使用三个定长顺序表：

| 顺序表 | 最大容量 | 主要字段 |
|---|---:|---|
| `BookList` | 200 | 编号、书名、作者、分类、库存、借阅次数 |
| `ReaderList` | 200 | 编号、姓名、性别、电话、身份 |
| `BorrowList` | 300 | 记录号、图书号、读者号、借阅日期、归还日期、截止日期 |

图书和读者通过借阅记录中的 `bookId`、`readerId` 建立关联。

## 编译运行

需要 GCC（Windows 可使用 MinGW）。

```bash
gcc -std=c11 -Wall -Wextra -pedantic -o library main.c book.c reader.c borrow.c fileio.c fileedit.c auth.c menu.c
```

Windows PowerShell：

```powershell
.\library.exe
```

Linux：

```bash
./library
```

程序首次运行时，会在当前目录自动创建三个空数据文件：

```text
book.txt
reader.txt
borrow.txt
```

## 数据文件格式

文件使用 UTF-8 文本，每行一条记录，字段由英文逗号分隔。

`book.txt`：

```text
bookId,bookName,author,category,stock,borrowCount
1001,数据结构,严蔚敏,计算机,5,12
```

`reader.txt`：

```text
readerId,readerName,gender,phone,role
2001,张三,男,13800000000,3
```

`borrow.txt`：

```text
borrowId,bookId,readerId,borrowTime,returnTime,deadline
3001,1001,2001,2026-06-15,未归还,2026-07-15
```

身份编号：`1` 为管理员，`2` 为教师，`3` 为学生。日期统一使用 `YYYY-MM-DD`。

## 使用流程

1. 启动程序，选择管理员、教师或学生身份。
2. 管理员可维护图书和读者资料，办理借还并编辑数据文件。
3. 教师和学生可查询图书、办理借还、查询个人记录和查看提醒。
4. 从主菜单退出时，内存中的数据会自动覆盖保存到三个 TXT 文件。

## 实现说明

- 新增记录前检查顺序表容量和编号唯一性。
- 删除记录时将后续元素向前移动。
- 图书和读者查询采用顺序查找，时间复杂度为 `O(n)`。
- 借阅排行复制图书表后使用冒泡排序，不改变原表顺序。
- 借阅成功后库存减一、借阅次数加一；归还成功后库存加一。
- 临期和超期提醒使用标准库日期转换计算实际天数差。

## 注意事项

- 文本字段中不要包含英文逗号，否则会影响数据解析。
- 请从程序主菜单正常退出，以确保数据保存。
- 当前登录模块用于课程演示，只区分身份，不包含账号和密码认证。
